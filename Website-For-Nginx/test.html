Instructions

The best scenario is using a Private Firefox window on a PC.

If using Firefox, a hard refresh (CTRL+SHIFT+R) will force both values to refresh. (CMD+SHIFT+R on Macs)

Other browsers will not refresh the TCP connection and may aggregate the TCP connection times.

If you wait about a minute other browsers should refresh the two values.
